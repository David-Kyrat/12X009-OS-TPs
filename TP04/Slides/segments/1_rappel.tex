\section{Rappel: But du TP}
\bframe{

    \begin{itemize}
      \item L'objectif général du TP est de créer un programme qui permet de faire de
       créer des \quo{verrou d'enregistrement interactif}.
      \item Le programme doit être capable poser un verrou et l'enlever sur une partie d'un fichier, de détécter la présence de verrous posés par les autres processus, le tout dans un mini \quo{shell} (CLI basique) 
    \end{itemize}

    Les principaux défis de ce TP sont:
    \begin{itemize}
        \item Comprendre les différents types de verrous et leurs implications
        \item Comprendre les différents arguments et flags possible à passer à \code{fcntl()} (file control)
    \end{itemize}
    
}
